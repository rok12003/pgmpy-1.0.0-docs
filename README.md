# pgmpy 1.0.0 Documentation

Offline-hosted API documentation for [pgmpy](https://github.com/pgmpy/pgmpy) version 1.0.0.

Built from the [v1.0.0 tag](https://github.com/pgmpy/pgmpy/tree/v1.0.0) using Sphinx.

## Why does this exist?

pgmpy 1.1.0 introduced breaking API changes and the official docs site (pgmpy.org) no longer serves 1.0.0 documentation. This site preserves the 1.0.0 docs for courses and projects that depend on that version.

## Install pgmpy 1.0.0

```bash
pip install pgmpy==1.0.0
```
